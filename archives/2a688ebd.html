机器学习--决策树

发表于2021-02-17|更新于2022-04-01|机器学习
|字数总计:594|阅读时长:2分钟|阅读量:

1.认识决策树

决策树的思想来源非常朴素，就像是if-then结果，例如：母亲给女儿介绍对象

女儿：多大年纪了？

母亲：26。

女儿：长得帅不帅？

母亲：挺帅的。

女儿：收入高吗？

母亲：不算很高，中等。

… …

2.原理

信息熵、信息增益等

2.1信息熵

比如有32支球队从里面猜冠军，错误就要付1块钱，要花多少钱可以知道谁是冠军。为了使代价最小，我们会使用二分法：

冠军在1-16号吗？… …依次询问，只需要5次就可以得到结果。

公式为：H(X) = P(X)logP(X)的全部合

但是，这样是有前提的，每支球队的夺冠几率相同，只要不满足这个条件，一定比5比特大。

总结：信息和消除不确定是相联系的。

我们得到的额外信息越多，那么我们猜测的代价越小，由此引入信息增益

2.2信息增益

特征A对训练集D的信息增益g(D,A)，定义为D的信息熵H(D)与特征A给定条件下D的信息熵H(D|A)之差，公式为：

g(D,A) = H(D) - H(D|A)

3.决策树的三种算法实现

  • ID3：信息增益最大的准则
  • C4.5：信息增益比最大的准则
  • CART：
    • 分类树：基尼系数最小的准则，在sklearn中可以选择划分的默认原则
    • 优势：划分更细致

4.API

  • class sklearn.tree.DecisionTreeClassifier(criterion=’gini’, max_depth=None,random_state=None)
    • 决策树分类器
    • criterion:默认是’gini’系数，也可以选择信息增益的熵’entropy’
    • max_depth:树的深度大小
    • random_state:随机数种子
  • 其中会有些超参数：max_depth:树的深度大小
    • 其它超参数我们会结合随机森林讲解

5.保存树的结构到dot文件（可选）

  • 1、sklearn.tree.export_graphviz() 该函数能够导出DOT格式
    • tree.export_graphviz(estimator,out_file=’tree.dot’,feature_names=[‘’,’’])
  • 2、工具:(能够将dot文件转换为pdf、png)
    • 安装graphviz
    • ubuntu:sudo apt-get install graphviz Mac:brew install graphviz
  • 3、运行命令
    • 然后我们运行这个命令
    • dot -Tpng tree.dot -o tree.png

6.决策树总结

  • 优点：简单的理解和解释，树木可视化。
  • 缺点：不能很好的推广数据过于复杂的树，被成为过拟合。
  • 改进：
    • 剪枝cart算法
    • 随机森林
